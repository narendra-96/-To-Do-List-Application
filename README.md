# To-Do List Web Application

## Description
This is a simple To-Do List web application developed using HTML, CSS, and JavaScript.
It allows users to add, delete, and mark tasks as completed.
Tasks are stored in browser local storage.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Local storage support

## How to Run
1. Open index.html in a web browser
2. Add tasks using the input field
3. Tasks will remain even after page refresh

## Conclusion
This project demonstrates basic front-end development concepts and DOM manipulation.
